# Week 1 - IP Addressing Basics Lab

This lab demonstrates basic IP addressing and connectivity between two PCs on the same subnet using Cisco Packet Tracer.

## Objectives
- Assign IP addresses to two PCs in the same /24 subnet  
- Test connectivity with ping commands  
- Understand private IP addressing and APIPA

## IP Scheme
- PC1: 192.168.1.10 /24  
- PC2: 192.168.1.11 /24  
- Gateway: None (direct connection)

## Verification
- Ping from PC1 to PC2 was successful  
- Screenshots and `.pkt` file included for reference
